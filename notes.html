Notes - Chapter 2: Polynomials

Introduction to Polynomials

A polynomial is an expression that consists of variables (also known as indeterminates) raised to whole-number powers and multiplied by coefficients. The general form of a polynomial in one variable is:

f(x) = anxn + an-1xn-1 + ... + a1x + a0

Where:

  • n is a non-negative integer (degree of the polynomial).
  • an, an-1, ..., a0 are constants (coefficients of the polynomial).
  • x is the variable (indeterminate).

Types of Polynomials

Polynomials can be classified based on their degree:

  • Constant Polynomial: Degree 0 (e.g., f(x) = 7).
  • Linear Polynomial: Degree 1 (e.g., f(x) = 2x + 3).
  • Quadratic Polynomial: Degree 2 (e.g., f(x) = x2 - 4x + 4).
  • Cubic Polynomial: Degree 3 (e.g., f(x) = x3 - 2x2 + 3x - 5).
  • Quartic Polynomial: Degree 4 (e.g., f(x) = x4 - x3 + x2 - x + 1).
  • Higher Degree Polynomials: Degree greater than 4 (e.g., f(x) = x5 - 2x4 + 3x3 - x2 + 4).

Polynomial Operations

You can perform various operations with polynomials, such as:

  • Polynomial Addition: Add polynomials by combining like terms.
  • Polynomial Subtraction: Subtract polynomials by subtracting like terms.
  • Polynomial Multiplication: Multiply polynomials using distributive property or FOIL method.
  • Polynomial Division: Divide polynomials using long division or synthetic division.

Examples

Let's take the following polynomials as examples:

  • f(x) = 2x3 + 4x2 - 5x + 6
  • g(x) = x2 - 3x + 2

When you add them, the result is:

f(x) + g(x) = 2x3 + 4x2 - 5x + 6 + x2 - 3x + 2 = 2x3 + 5x2 - 8x + 8

Important Theorems and Properties

  • Factor Theorem: If f(a) = 0, then (x - a) is a factor of the polynomial f(x).
  • Remainder Theorem: When a polynomial f(x) is divided by (x - a), the remainder is f(a).
  • Fundamental Theorem of Algebra: Every non-constant polynomial equation has at least one complex root.

Graphing Polynomials

The graph of a polynomial function is continuous and smooth. The degree of the polynomial determines the number of turning points in the graph.

Applications of Polynomials

Polynomials are widely used in various fields including physics, engineering, economics, and computer science. They are used to model growth rates, optimize functions, and describe physical phenomena.
